# BuySafe Guarantee module for Magento 2

BuySafe Guarantee module for Magento 2.

You can:
- Show BuySAFE seal at all pages;
- Show BuySAFE guarantee at checkout success page

# Installation
## Composer

```bash
composer require 28software/module-buysafe-guarantee
```
